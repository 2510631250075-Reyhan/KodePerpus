SISTEM MANAJEMEN PERPUSTAKAAN DIGITAL

Deskripsi Project

Sistem Manajemen Perpustakaan Digital adalah sistem berbasis Java yang dibuat untuk membantu pengelolaan data buku perpustakaan secara lebih terstruktur dan efisien.
Project ini dibuat dengan menerapkan konsep struktur data dan algoritma seperti searching, sorting, CRUD, dan penyimpanan data file.

Fitur Utama

1.Menambahkan data buku
2.Menampilkan daftar buku
3.Mengedit data buku
4.Menghapus data buku
5.Searching data buku
6.Sorting data buku
7.Pengelolaan status buku
8.Penyimpanan data ke file

Cara Menjalankan Program

1. Clone repository ini
2. Buka project di VSCode
3. Compile file Java
4. Jalankan program melalui terminal atau IDE
5. isi sesuai fitur yang dipilih
